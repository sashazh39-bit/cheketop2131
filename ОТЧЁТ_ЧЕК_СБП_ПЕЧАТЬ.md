# Отчёт: чек СБП с печатью

## Параметры

| Параметр | Значение |
|----------|----------|
| **Формат** | СБП (Квитанция о переводе по СБП) |
| **Донор** | donors/07-03-26_04-34 (1).pdf |
| **Выходной файл** | чеки 08.03/чек_сбп_с_печатью_final.pdf |
| **Плательщик** | Сидоров Алексей Петрович |
| **Печать** | Круглая красная (ВТБ), позиция (80, 30) |
| **Document ID** | Заменён на уникальный |

## Этапы генерации

1. **Патч полей** — `sbp_full_toolkit.py patch`
   - Замена payer → «Сидоров Алексей Петрович»
   - CID-кодирование (без чёрных полос)
   - Сохранение структуры PDF

2. **Добавление печати** — `add_stamp.py`
   - Создание круглой красной печати (PIL)
   - Наложение на страницу (PyMuPDF)

## Команды

```bash
# 1. Генерация чека СБП
python3 sbp_full_toolkit.py patch "donors/07-03-26_04-34 (1).pdf" "чеки 08.03/чек_сбп_с_печатью.pdf" \
  --payer "Сидоров Алексей Петрович" --patch-id

# 2. Добавление печати
python3 add_stamp.py "чеки 08.03/чек_сбп_с_печатью.pdf" "чеки 08.03/чек_сбп_с_печатью_final.pdf"
```

## Результат

- Чек в формате СБП (ВТБ)
- Печать добавлена
- CID-патч без искажений
- Document ID заменён
