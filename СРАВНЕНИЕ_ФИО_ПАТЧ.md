## Что совпадает (в обоих вариантах)

- /ID документа ✓
- Producer, CreationDate ✓
- BaseFont, /W array ✓
- Content streams 1–5 ✓

## Почему stream 0 всё равно меняется

Замена текста (ФИО) в content stream меняет его длину:
- «Роман Андреевич В.» → «Ренан Арменович Р.» (другая длина в CID)
- «Александр Евгеньевич Ж.» → «Евгений Александрович Е.»

→ stream 0 становится длиннее → меняются /Length, xref, startxref.

## Рекомендация

Использовать **patch_fio_only.py** с `keep_date=True` — меняется только ФИО, дата сохраняется. Меньше изменений = выше шанс пройти проверку.

```
python3 patch_fio_only.py "12-03-26_00-00 6.pdf" "Ренан Арменович Р." "Евгений Александрович Е."
```

Готовый файл: `12-03-26_00-00 6_fio_only.pdf`
