# Telegram-бот для замены суммы в чеках

Бот изменяет сумму в PDF-чеках (Альфа-Банк, ВТБ), сохраняя структуру и метаданные файла.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

1. Создайте бота через [@BotFather](https://t.me/BotFather).
2. Задайте переменную окружения с токеном:

```bash
export TELEGRAM_BOT_TOKEN=your_bot_token
python bot.py
```

## Использование

1. Отправьте PDF-чек боту.
2. Выберите банк: **Альфа-Банк** или **ВТБ**.
3. Введите сумму: с какой на какую менять (например: `10 5000` или `50 10000`).
4. Получите готовый PDF.

## Поддерживаемые форматы

- **Альфа-Банк** — чеки с форматом CID без квадратных скобок.
- **ВТБ** — чеки с форматом `[(\x00\x18)...(\x04@)]`.

Если формат не определился автоматически, выберите банк вручную.

## Выписка Альфа «с нуля» (`bot_standalone.py`)

В корне репозитория — **`выписка_жуков_177414_СБП_одна_операция.pdf`** (шаблон выписки Альфа для бота). После `git pull` он будет на сервере. Пересобрать: `python3 build_statement_zhukov_177414.py`.
Запускайте **`python3 bot_standalone.py`**, не `bot.py` — в `bot.py` другой поток выписок.
