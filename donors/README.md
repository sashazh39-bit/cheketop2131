# Папка донорских чеков

Поместите сюда PDF-чеки, из которых можно копировать корректно отрендеренные блоки текста (TJ + Tm).

**Использование:**
- Чек с нужным ФИО (например «Андрей Максимович Р.») — для копирования блока плательщика
- Чек с нужным банком («Сбербанк») — для копирования блока банка
- Чек с нужным телефоном — для копирования блока телефона

**Важно:** Донор должен быть того же формата ВТБ/СБП, что и целевой чек (одинаковый CMap, шрифты).

Скрипт `patch_from_donor.py` (если будет добавлен) сможет извлекать TJ-блоки из доноров и подставлять их в целевой чек, сохраняя корректное отображение и Tm.
