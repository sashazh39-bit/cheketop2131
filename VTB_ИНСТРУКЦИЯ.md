# Чек ВТБ — настройка значений и выравнивание

## Как менять значения

1. Отредактируй **`vtb_config.json`**:

```json
{
  "date": "now",
  "payer": "Евгений Александрович Е.",
  "recipient": "Анна Петрова С.",
  "amount": 10000,
  "update_id": true,
  "output_folder": "10.03"
}
```

2. Запусти:

```bash
python3 vtb_patch_from_config.py input.pdf
```

3. Результат в папке `10.03/` (или из `output_folder`).

---

## Параметры конфига

| Поле | Описание |
|------|----------|
| `date` | `"now"` — текущие дата и время; или `"10.03.2026, 15:30"` |
| `payer` | ФИО плательщика (например, «Иван Петрович П.») |
| `recipient` | ФИО получателя |
| `amount` | Сумма в рублях (число) |
| `update_id` | `true` — обновить /ID документа |
| `output_folder` | Папка для сохранения PDF |

---

## Выравнивание

Правый столбец выравнивается по вертикальной «стенке» (WALL).

Калибровка в `vtb_test_generator.py`:

```python
PTS_CALIBRATION = {
    "date": 4.40,
    "payer": 4.57,
    "recipient": 4.55,
    "amount": 6.68,
}
```

Для коротких имён выравнивание может слегка сбиваться; для длинных имён — близко к эталону.

---

## CMap (символы)

Поддерживаются: кириллица (А–Я, а–я), цифры, пробел, точки, запятые, ₽.

Если символ не найден, будет ошибка — добавь его в `vtb_cmap.py`.

Поле **ID операции в СБП** не меняется.
