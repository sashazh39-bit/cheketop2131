# Команды для работы с чеками ВТБ СБП

## Проверка доступных заглавных букв

Перед генерацией чека проверьте, какие буквы в ФИО будут заглавными:

```bash
python3 add_glyphs_to_13_03.py \
  --replace --hybrid-safe --auto-base --check-caps \
  --payer "Плательщик Иванович И." \
  --recipient "Получатель Алексеевич П."
```

**Результат:** список заглавных и строчных букв для данного ФИО.

---

## Генерация чека

```bash
python3 add_glyphs_to_13_03.py \
  --replace --hybrid-safe --auto-base \
  --payer "Имя Отчество Ф." \
  --recipient "Имя Отчество Ф." \
  --bank "Т-Банк" \
  --amount СУММА \
  --date "ДД.ММ.ГГГГ" \
  --time "ЧЧ:ММ" \
  --account "XXXX" \
  -o чек.pdf
```

### Параметры:

| Параметр | Описание | Пример |
|----------|----------|--------|
| `--payer` | ФИО плательщика | `"Алексей Иванович П."` |
| `--recipient` | ФИО получателя | `"Никита Петрович С."` |
| `--bank` | Банк получателя | `"Т-Банк"`, `"Сбер"`, `"Альфа"` |
| `--amount` | Сумма в рублях | `10000` |
| `--date` | Дата перевода | `"13.03.2026"` |
| `--time` | Время перевода | `"17:13"` |
| `--account` | Последние 4 цифры счёта | `"9426"` (по умолчанию из шаблона) |
| `-o` | Имя выходного файла | `чек.pdf` |

## Примеры

### Простой чек:
```bash
python3 add_glyphs_to_13_03.py \
  --replace --hybrid-safe --auto-base \
  --payer "Алексей Михайлович Л." \
  --recipient "Магомед Алексеевич С." \
  --bank "Т-Банк" \
  --amount 10000 \
  --date "13.03.2026" \
  --time "17:13" \
  -o чек_готов.pdf
```

### Чек с другим номером счёта:
```bash
python3 add_glyphs_to_13_03.py \
  --replace --hybrid-safe --auto-base \

  -o чек_готов.pdf
```

## Поддерживаемые буквы

Все буквы русского алфавита поддерживаются.  
Некоторые сложные заглавные (Ф, Ч, Ю, М, К, Н, Я, И и др.) требуют "слотов" для инъекции глифа.

**Ограничение слотов:** доступно 3 слота (Е, Ж, Г).  
Если в ФИО больше 3 сложных заглавных — часть останется строчной.  
Используйте `--check-caps` чтобы проверить заранее.

---

## Частые ошибки

- **Буква строчная вместо заглавной** — не хватило слотов. Упростите ФИО или уберите редкие заглавные.
- **Чек не прошёл проверку** — попробуйте запустить команду ещё раз (генерируется новый Document ID).
- **Размер файла > 9.2 КБ** — используется другой базовый шаблон; попробуйте снова.
