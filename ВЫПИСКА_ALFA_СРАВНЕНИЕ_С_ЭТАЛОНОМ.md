## Вывод для проверки бота

- **Стабильная структура** (как у ВТБ-чеков относительно донора): те же `BaseFont`, совпадение декомпрессии ключевых content streams, неизменённые ToUnicode для дополнительных шрифтов.
- **Плановые отличия:** новый `/ID`, новый `ModDate`, расширение **первого** ToUnicode и пересборка одного короткого zlib-потока при том же декодированном содержимом.
- Код операции **`OST1_…`**: заменяется через перебор CMap в `patch_replacements`; новый код использует только символы из исходной строки (иначе нет записей в том же CMap).
