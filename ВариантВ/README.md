# Вариант B — компактный чек из check(3)

Трансформация чека `check (3).pdf` в компактный формат (без «Сообщение», MediaBox 437 pt).

## Шаги

### 1. Создать compact_base.pdf

```bash
cd ..
python3 ВариантВ/transform_to_compact.py "база_чеков/vtb/СБП/check (3).pdf" ВариантВ/compact_base.pdf
```

### 2. Генерация чека с нужными данными

```bash
python3 ВариантВ/gen_receipt.py receipt.pdf --payer "Максим Андреевич Б." --recipient "..." --amount 8700
```

## Что делает transform_to_compact.py

- Удаляет блок «Сообщение» (метка + 2 строки значений)
- Объединяет «Имя» и «плательщика» в одну строку (где возможно)
- Меняет MediaBox: высота 437.25 pt (как в random_receipt_2)

## Ограничения

- Объединение «Телефон получателя» зависит от структуры PDF
- Размер файла ~11 KB (цель 8–10 KB требует дополнительной оптимизации)
